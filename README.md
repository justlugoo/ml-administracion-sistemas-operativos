# ML — Administración de Sistemas Operativos

Parcial aplicado en Python usando Machine Learning sobre datos de sistemas operativos.

## Contenido

| Punto | Tema | Técnica |
|-------|------|---------|
| 1 | Exploración de datos | pandas |
| 2 | Clasificación de instalación de SO | Árbol de Decisión |
| 3 | Predicción de falla de disco | Árbol de Decisión |
| 4 | Comparación de modelos | Árbol vs Random Forest |
| 5 | Agrupamiento de equipos | K-means + PCA |

## Tecnologías
Python 3 · pandas · scikit-learn · matplotlib · seaborn · openpyxl

## Ejecución
Desarrollado en Google Colab. Abrir el archivo `.ipynb` y ejecutar celda por celda.

## Datos
`base_datos_parcial_ml_so.xlsx` — 3 hojas: `instalacion_SO`, `falla_disco`, `clusters_laboratorio`.
